DESKTOP DESIGN GUIDELINES (1440 × 768)

FRAME & LAYOUT
Frame: 1440 × 768.
Use a 12-column grid with 80px left and right margins and 24px gutter.
Avoid vertical scrolling. Place key content within the top 600px.
Canvas: Desktop 1440 × 768. Keep key content inside top 600px.

Grid: 12-column, 80px side margins, 24px gutter.

Role Selection screen: display 4 cards (2 columns × 2 rows) inside the top 600px. Each card must show Role name (16px Medium), short description, and a “Select” button.

Clicking a role: produce a separate Login screen for that role. The Login screen must display:

Role title at top center (e.g., “Admin”) — 20px Semibold recommended.

Subtitle: role short description or selected user name (optional).

Password input field immediately below the title/subtitle (40px height, label above).

Login primary button below the password field (40px height).

Keep Login screen compact — no vertical scrolling. Place all content within top 600px.
